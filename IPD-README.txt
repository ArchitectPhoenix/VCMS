The files IPD-rand.csv and IPD-fix.csv are the data for the SP and FP treatments, respectively.  In both of these files, these columns will be found:


round: the round of the game at that moment, 1-100
player: ID of the player, it contains the experimental session they played.
action_player: response of the player, C: Cooperate, D: Defect.
opponent: ID of the opponent.
action_opponent: response of the opponent, C: Cooperate, D: Defect.
payoff: rewards for their actions in the IPD game. R = 3, T = 4, S = 0, P = 1.
time_php and time_js: time the player took to respond, not used.
treatment: "fix" for FP and "rand" for SP
session: experimental session, not used.
player_id: another ID of the player, generated by the system.
prev: actions of the previous round. Format: player/opponent.
context: previous action + current action
prev_player: previous action of the player
prev_opp: previous action of the opponent.



The file "Instructions of the experiment.docx" provides the full instructions the players received to play the game, this was explained beforehand.
